Easy Kanban
===========
[TOC]

v0.9.0 - Jan 2026

A simple and easy to use Kanban board application built with web technologies. Just born to handle clients and steps, configure the data json files and start to use it!

# Getting Started

```bash
cd easy-kanban
npm install
npm run start
```
